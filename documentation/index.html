Technical documentation of Todolistme

The architecture of this application is based on the MVC model. The pattern is composed of three types of modules with three different responsibilities: models, views and controllers.

  • A Model contains the data to display.
  • A View contains the presentation of the graphical interface.
  • A Controller contains the logic concerning the actions performed by the user.
